# Codex Port

Codex uses `.codex-plugin/plugin.json` for plugin metadata and `.codex/commands/transcend.md` for the `/transcend` command suite.

Role providers are configured in `transcend.yaml`. Use Codex skills, MCP tools, command prompts, or manual fallbacks to fill roles.

Run `/transcend bootstrap` after install. It should install or report the Codex providers the host supports. Run `/transcend doctor` before the first full workflow.
